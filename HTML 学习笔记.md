# HTML标签

## 一.定义

通过一系列的==标签==（也 称为元素）来定义文本、图像、链接等等。HTML标签是==尖括号==包围的==关键字==。

### 2.双标签：绝大多数都是双标签

标签通常成对出现，包括开始标签和结束标签，内容位于这两个标签之间。

~~~
段落标签:<p>这是一个段落。</p>
标题标签:<h1>这是一个标题。</h1>
超链接标签:<a href="#">这是一个超链接</a>
~~~

### 2.单标签:

没有结束标签

~~~
input标签:<input type="text">
换行标签:<br>
分割线标签:<hr>
~~~

区别:单标签用于没有内容的元素，双标签用于有内容的元素

## 3.标签不区分大小写，但推荐小写，因为小写更规范。

## 4.标签之间的关系：并列关系，嵌套关系，可以使用Tab键进行缩进。



## 二.HTML属性

1. 用于描述元素的特征和行为，提供附加信息。
2. 可以写在：==起始标签==  或  ==单标签==中：形式如下

~~~http
<标签名 属性名=“属性值” 属性名=“属性值”>可以是单引号也可以是双引号
例如
<marquee loop=:“1” bgcolor=“orange”>让天下没有难学的技术</marquee>
<input type="password">
~~~

3. 有些特殊的属性，没有属性名，只有属性值，例如：

` <input disabled> `

4.注意点：

~~~http
不同的标签，有不同的属性；也有一写通用属性（在任何标签内都能写，后面会详细总结）
属性名、属性值不能乱写，，都是W3C规定好的。
属性名，属性值，都不区分大小写，但推荐小写。
双引号，也可以写成单引号，甚至不写都行，但还是推荐双引号。
标签中不要出现同名属性，否则后面写的会失效，例如
<input type="text" type="password">
~~~

## 三.文件结构

1. 在网页中，如何查看某段结构的具体代码? —点击鼠标右键，选择检查。
2. 【检查】和【查看网页源代码】的区别

~~~
【查看网页源代码】看到的是：程序员编写的源代码。
【检查】看到的是：经过浏览器“处理”后的源代码。

~~~

~~~markdown
想要呈现在网页中的内容写在==body标签==中

==head==标签中的==title==标签可以指定网页中的标题
图示
<html>

	<head>
		<title>.......<title>
	</head>
	<body>
   		 ......
	</body>

</html>
~~~

## 六.注释

1. 特点：注释的内容会被浏览器所忽略，不会呈现到页面中，但源代码中依然可见。
2. 作用：对代码进行解释和说明。
3. 写法

~~~html
<!-- 下面的文字只能滚动一次 -->
<marquee loop="1">
尚硅谷
    <input type="text">
</marquee>
<!-- 下面的文字可以无限滚动 -->
<marquee>
    尚硅谷123
</marquee> 

~~~

4. 注释不可以嵌套，反例

~~~html
<!-- 
这是一段注释
    <!----这是一段注释 -->
-->

~~~

5. 快捷键：Crtl+/

## 七.文档声明

1. 作用：告诉浏览器当前网页的版本。
2. 写法：

* 旧写法：要依网页 所用的HTML版本而定，写法有很多。

  ~~~
  具体有哪些写法请参考：W3C官网声明。
  ~~~

* 新写法：推荐使用HTML5的写法。

~~~
<! DOCTYPE html>
~~~

3. 注意：文档声明必须在网页的第一行，且在==html==标签的外侧。

## 八.字符编码

1. 计算机对数据的操作：

* 存储时，对数据进行==编码==。
* 读取时，对数据进行==解码==。

2. 编码，解码会遵循一定的规范—–**字符集**。

3. 字符集有很多种，常见的有（了解）：

~~~
ASCII
ISO 8859-1
GB2312
GBK
UTF-8：包含世界上所有语言的
~~~

4. 使用原则是怎样？

~~~markdown
原则1.存储时，务必采用合适的字符编码
==否则：无法存储，数据会丢失==
~~~

我们一般采用UTF-8

5. 总结：

* 平时编写代码时，统一采用==UTF-8==进行编码（最稳妥）。
* 为了让浏览器在渲染==html==文件时，不犯错误，可以通过==meta==标签配合==charset属性指定字符编码。

~~~html
<! DOCUTYPE html>
<html>
    <head>
        <meta charset="UTF-8"/>
        <body>
            <!-- 下面的内容只滚动一次 -->
            <marquee loop="1">
            尚硅谷
            </marquee>
        </body>
    </head>
</html>
~~~
