# HTML标签

## 一.定义

1.通过一系列的==标签==（也 称为元素）来定义文本、图像、链接等等。HTML标签是==尖括号==包围的==关键字==。

### 2.双标签：

标签通常成对出现，包括开始标签和结束标签，内容位于这两个标签之间。

~~~
段落标签:<p>这是一个段落。</p>
标题标签:<h1>这是一个标题。</h1>
超链接标签:<a href="#">这是一个超链接</a>
~~~

### 2.单标签:

没有结束标签

~~~
input标签:<input type="text">
换行标签:<br>
分割线标签:<hr>
~~~

区别:单标签用于没有内容的元素，双标签用于有内容的元素

## HTML文件结构

~~~html
<!-- 这里放置的文档的元信息 -->
<！DOCTYPE html>   
<html>
<head>
<!-- 这里放置文档的元信息 -->
    <title>文档标题</title>
    <meta charset ="UTF-8">
    <!-- 连接外部样式表或脚本文件等 -->
    <link rel="stylesheet"type="text/css"href="styles.css">
</head>
    <body>
        <!-- 这里放置页面内容 -->
        <h1>这是一个标题</h1>
        <p>这是一个段落</p>
         <a href="https://www.example.com">这是一个链接</a>
        <！--其他内容 -->
        </p>
        </h1>
    </body>
</html>
</i!-->
~~~

* 1、2行告诉浏览器这是一个HTML文件
* 3、19<html>标签对：HTML文档的起始点，也是文档的最外层容器。
* 4、18行<head>标签对，文档的头部，包含了一些文档的原信息如
  如标题，文档的样式表，编码格式，CSS，JS文件
* 11、19行<body>标签对包含实际在浏览器中显示页面的内容

像文本，图像，链接等等，否则在浏览器中就无法看到。
