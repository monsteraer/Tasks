#                               CSS笔记

## 一.CSS简介

1. HTML：结构
2. CSS：表现
3. JavaScript:行为

* CSS的全称：层叠样式表（Cascading Style Sheets）

层叠:一层一层涂上去；表：所要干什么的任务表格。

* CSS也是一种标记语言，用于给HTML结构设置样式，例如文字大小，样式，元素宽高等。

~~~markdown
简单理解：==CSS==可以美化==HTML==,让==HTML==变得更漂亮。
核心思想：==HTML==搭建结构，==CSS==添加样式，实现了：结构与样式的分离。
~~~

## 二.CSS行内样式

1. 写在标签的==style==属性中（又称内联样式）
2. 语法：

~~~html
<h1 style="color:red;front-size:100px;">欢迎来到这里</h1>
~~~

3. 注意点：

   * style的属性值不能随便写，写要符合CSS的语法规范，是：==名：值；==的形式。

   * 行内样式表只能控制当前的标签的样式，对其他的标签无效。

4. 存在的问题：

书写繁琐，样式不能复用，并且没有体现出：==结构与样式分离的思想==看举的例子，h1里面的结构与样式混在一起，不推荐大量使用，只有对当前元素添加简单样式时才偶尔使用。
